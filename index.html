Back-End Development Course

This course is designed to introduce students to the concepts and practices of backend web development. Students will learn how to design, build, and deploy web applications using various tools and frameworks. Students will also learn how to create and consume web services and APIs, as well as how to secure, test, and optimize their web applications.

Learning Objectives

  • Understand the role and responsibilities of a backend web developer
  • Learn how to use common backend web development tools and frameworks
  • Develop web applications that interact with databases, web servers, and web clients
  • Create and consume web services and APIs using RESTful and GraphQL approaches
  • Implement authentication, authorization, encryption, and error handling in web applications
  • Test, debug, and optimize web applications for performance and scalability
  • Deploy web applications to cloud platforms and services
